Мой главный проект для изучения программирования и портфолио. Со временем я буду его улучшать, изучая новые технологии и интегрируя их в свой блог.
Сейчас в блоге есть посты, теги, админка с моделями и админ-панель для управления объектами, стиль сайта выполнен с помощью bootstrap/css, есть api на django rest framework(для просмотра и редактирования пользователей и групп).


Для запуска сервера:
1. virtualenv venv
2. venv\Scripts\activate
3. pip install -r requirements.txt
4. blogengine/settings.py # Обратить внимание на  значения "DATABASES" и вписать собственные.
5. py manage.py runserver
